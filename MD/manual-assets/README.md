## 목적

이 폴더는 `MD/manual`의 웹 매뉴얼에서 참조하는 화면 캡처와 이미지 자산을 보관한다.

## 파일 역할

| 파일 | 역할 |
|------|------|
| `first-page-after-login.png` | 로그인 후 첫 화면을 전체 브라우저 기준으로 캡처한 원본 이미지다. 첫 화면 매뉴얼의 설명 이미지로 사용한다. |

## 관리 기준

- 화면 캡처는 필요한 경우 전체 브라우저 화면 기준으로 남긴다.
- 웹 매뉴얼에서 재생성 가능한 중간 산출물은 원칙적으로 보관하지 않는다.
- 같은 화면을 다시 캡처하면 관련 웹 매뉴얼의 설명 위치와 강조 번호도 함께 확인한다.
