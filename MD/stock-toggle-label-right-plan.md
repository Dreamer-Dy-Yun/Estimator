# 재고·오더 토글 위치 미세조정 계획

- 작성일자: 2026-04-15
- 변경일자: 2026-04-15
- 지시자: Yun Daeyoung

## 목적
- `안전재고 직접입력` 레이블의 오른쪽에 토글 스위치가 오도록 배치 순서를 조정한다.

## 영향 범위
- `dashboard-app/src/dashboard/components/product-secondary/cards/StockOrderCard.tsx`

## 수정 방식
- 제목 우측 라벨 내부 렌더 순서를 `텍스트 -> 토글 input -> 토글 트랙` 순서로 변경한다.
- 기존 체크 상태/변경 이벤트 로직은 유지한다.

## 검증 포인트
- 화면에서 `안전재고 직접입력` 텍스트 오른쪽에 토글이 표시되는지
- 토글 ON/OFF 동작이 기존과 동일한지
