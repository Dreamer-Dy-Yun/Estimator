# 오더 후보군 상세 팝업 -> 페이지 전환 계획

- 작성일자: 2026-04-21
- 변경일자: 2026-04-21
- 지시자: Yun Daeyoung (사용자)

## 목표
- 오더 후보군 카드 클릭 시 뜨는 상세 팝업을 제거하고, 페이지 기반 상세 화면으로 재작성한다.
- 기존 디자인 톤(공통 카드/필터 스타일)을 유지한다.

## 영향 범위
- `dashboard-app/src/App.tsx`
  - 상세 경로 라우트 추가 (`/dashboard/snapshot-confirm/:stashUuid`)
- `dashboard-app/src/dashboard/pages/SnapshotConfirmPage.tsx`
  - 팝업 오버레이 로직 제거
  - 카드 클릭 시 상세 라우트로 이동
  - 상세 화면 페이지 본문 렌더링(검색/정렬/리스트/액션 유지)
- `dashboard-app/src/dashboard/pages/SnapshotConfirmPage.module.css`
  - 오버레이 스타일 제거/완화
  - 상세 페이지 레이아웃 클래스 추가

## 구현 포인트
1. 목록 페이지와 상세 페이지를 동일 컴포넌트 내 라우트 파라미터로 분기한다.
2. 상세 페이지에서 뒤로가기(목록) 액션을 제공한다.
3. 삭제 완료 시 상세 대상이 삭제되면 목록 페이지로 이동한다.

## 검증 체크
- 카드 클릭 시 모달이 아니라 URL 변경과 함께 상세 페이지가 뜨는지 확인
- 브라우저 뒤로가기로 목록 복귀 가능한지 확인
- 상세 페이지 검색/정렬/삭제 동작 정상 확인
