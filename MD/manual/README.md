## 목적

이 폴더는 HAN.A Estimator 사용자 매뉴얼을 웹 화면 단위로 작성하는 원본을 보관한다. PPT는 별도 원본으로 관리하지 않고, 필요할 때 웹 매뉴얼의 각 화면을 전체 브라우저 기준으로 캡처해 이어 붙인다.

## 파일 역할

| 파일 | 역할 |
|------|------|
| `HAN_A_first_page_manual.html` | 로그인 이후 첫 화면을 설명하는 웹 매뉴얼 원본이다. 1280×720 단위의 `section.slide`를 세로로 쌓는다. |
| `first-page-manual.css` | 첫 화면 웹 매뉴얼의 화면 크기, 캡처 모드, 카드/라벨/강조 번호 스타일을 담당한다. |
| `../manual-assets/first-page-after-login.png` | 로그인 후 첫 화면 전체 브라우저 캡처 원본이다. 설명용 확대·강조 화면의 기준 이미지로 사용한다. |

## 캡처 기준

- 기본 캡처 기준은 전체 브라우저 화면이다.
- 각 장은 1280×720 크기의 `section.slide` 하나로 만든다.
- 전체 매뉴얼을 훑을 때는 `HAN_A_first_page_manual.html`을 그대로 열며, 첫 화면 설명 장부터 보이게 한다.
- 커버가 필요하면 `HAN_A_first_page_manual.html?slide=1`로 별도 캡처한다.
- 특정 장만 캡처할 때는 `HAN_A_first_page_manual.html?slide=2`처럼 `slide` 파라미터를 사용한다.
- PPT가 필요해지면 필요한 장을 브라우저에서 캡처해 순서대로 붙인다.

## 확장 원칙

- 실제 앱 화면에 붙이지 않고 문서 영역에서 독립적으로 관리한다.
- 앱 코드에 도움말 라우트나 메뉴로 연결하는 작업은 별도 요구가 있을 때만 진행한다.
- 새 화면을 추가할 때는 같은 1280×720 화면 단위를 유지하고, 이미지 자산은 `MD/manual-assets`에 둔다.
