# DONE 폴더 안내

이 폴더는 완료된 문서/작업 메모를 보관합니다.

## 파일명 규칙
- `YYYY-MM-DD-작업요약.md`
- 예: `2026-05-13-dashboard-api-정리.md`

## 완료 항목 기준
- TODO에서 모든 체크리스트가 완료된 항목만 이동
- 최종 상태, 최종 결정사항, 참고 링크/이슈번호를 남김

## 완료 템플릿(예시)

```markdown
# 작업명

- 상태: DONE
- 완료일:
- 작업자:
- 관련 링크/이슈:

## 최종 결과
- 

## 변경/결정 사항
- 

## 후속 조치
- 없음 / 있음(링크)
```
