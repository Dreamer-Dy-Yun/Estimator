# TODO 작업 폴더 안내

이 폴더는 진행 중이거나 아직 처리되지 않은 작업을 관리합니다.

## 파일명 규칙
- `YYYY-MM-DD-작업요약.md`
- 예: `2026-05-13-dashboard-api-정리.md`

## TODO 템플릿

```markdown
# 작업명

- 상태: TODO / IN-PROGRESS / BLOCKED
- 작업자:
- 시작일:
- 목표 완료일:

## 배경
- 

## 작업 내용
- [ ] 

## 확인 항목
- [ ] 

## 비고
- 
```

## 이동 규칙
- 완료된 항목은 같은 파일명으로 `MD/DONE`으로 이동합니다.
- 진행 중인 작업은 해당 항목 내에서 최신 상태를 최신화합니다.
